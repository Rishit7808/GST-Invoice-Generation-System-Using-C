# GST Billing System

This is a simple GST Billing System developed in C. The program allows the user to enter item details such as item name, GST percentage, quantity, and unit price. It then calculates CGST, SGST, gross amount, and generates a complete invoice.

The invoice is displayed on the screen and is also saved to a text file (`bill.txt`) for future reference.

## Features

- Enter multiple items
- Calculate CGST and SGST automatically
- Generate item-wise GST invoice
- Calculate grand total
- Display invoice on screen
- Save invoice to a file

## Concepts Used

- Structures
- Functions
- Arrays
- File Handling
- User Input and Output

## How to Run

1. Compile the program:
```

gcc cp1.c -o billing

```

2. Run the executable:
```

./billing

```

## Output

The program generates a GST invoice showing:
- Item Name
- GST Percentage
- Quantity
- Unit Price
- CGST
- SGST
- Gross Amount
- Grand Total

A copy of the invoice is also stored in `bill.txt`.
